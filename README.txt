                                                PROJECT RBNB (Refugee Booking N' Boys)


1) Le but ? 

Réalisation d'un site, sur la disponibilité d'aide aux logements pour les personnes réfugiés en france.
Pour une recherche centralisée et répertoriée sur le territoire et donc par ville.

 


2) Ce que je peux faire sur ce site ? 

On peut rechercher un logement par zones, par departements, par villes.

Possibilité au propriétaire de logement disponible, de ce connecter afin de poster une annonce avec photos du
bien, pour ensuite permettre à un utilisateur d'effectuer une recherche précise avec les informations du 
proprietaire pour le rappeler via son numéro de téléphone mobile ou autres sur la page même.

 


3) Qui peut accéder au site ? 

Seul le proprietaire à la possibilité de crée un compte afin de poster une annonce de son bien.

Les utilisateurs peuvent seulement naviguer sur le site comme vitrine et leurs permettent juste de prendre l'info
sur les logements ou appartements disponibles.

  


4) Bdd ? 

il y aura bien-évidemment une base de données, pour pouvoir stocker les informations personnelles des personnes, 
aussi pour attribuer un rôle de connection pour poster une annonce.





 
5)  Hébergeur ? 

Le site sera qu'un exercice de stage ou un prototype, mais par la suite pourrait servir de base ou d'idée pour
un véritable site !!! l'hebergeur pour teste serait eventuellement sur AWS !!! A suivre.

 


6) Améliorer la visibilité ? 

Le SEO sera mis en place pour aider à la visibilité et le partage de liens sociaux qui sera sur le site 
à disposition.

       


7)  Idée d'amélioration ?

A voir avec mes tuteurs de JustAI

                                           Descriptif du cahier des charges



Projet : Pouvoir proposer à des propriétaires / structures qui possèdent des logements pour des réfugiés 
mais qui ont peut de moyen financiers.


En tant que propriétaire je peux :

-Créer une annonce pour un logement contenant :
-Des photos du logement
-Une description
-L’adresse du logement (département et libellé)
-Une adresse email de contact
-Un numéro de téléphone de contact (mobile ?)
-Un lien vers le site du propriétaire (si existant)


En tant que agence je peux :

-Voir les annonces existantes qui contiennent en visuel :
-Des photos du logement
-Une description
-Un bouton / lien adresse email de contact
-Un numéro de téléphone de contact (mobile ?)
-Un lien vers le site du propriétaire (si existant)


Filtre les annonces par :

-Libellé
-Région
-Département



Le site internet devra :

-Etre développé en PHP à l’aide du framework Symfony
-Posséder une base de données MySQL
-Posséder un thème CSS Responsive (utilisant Bootstrap éventuellement)
-Permettre la création d’un compte en tant que “propriétaire de logements”
-Posséder une barre de recherche pour trouver les annonces


Il est demandé de produire en tant que livrable intermédiaire (en plus du livrable final) :
Un wireframe / Maquette du site
Un diagramme UML de classes SQL
